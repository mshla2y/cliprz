Hello folder

To create a new project you must create all files in ./application/project/ as in example i want to create a project named test, So i create a folder in ./application/project/hello


Write a view

Create a new file that name as in example hello, please do not forget this rules to create a view file :

Rules
  • View file must end with .view.php as in example hello.view.php
  • View file name can contain only letters, numbers, periods (.), hyphens (-), and underscores (_). It can't contain special characters, accented letters, or letters outside the Latin alphabet.
  • View file name case not sensitive.
Now create hello.view.php in ./application/project/hello/
