Cliprz framework

The Cliprz program is a free PHP framework open source written in PHP version 5.3.0 or later. It helps you create a project without needing to insert PHP big codes. What distinguishes Cliprz Framework is that it is easy, quick, simple and secure. There you can route URLs without using controller / functions / parameters. Additionally, you can use controller MySQL databases (MySQL easy system). One easy way to create a quick admin panel (scaffold) system with full control pages. And it can help you with plugins, library and more!
