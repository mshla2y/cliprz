Preface

What is Cliprz

It's a free and open source PHP framework written in PHP version 5.3.0 or higher, This framework help you to create a PHP project without writing big codes, It's easy, fast, simple and secure, What distinguishes Cliprz framework You can routing URLs without using controller/functions/parameters, You can use MySQL database driver object (easy MySQL system), Easy way to create fast admin panel (scaffold) system with full pages control, Help plugins and libraries, That's not all there is more.


Requirements

You need PHP 5.3.0 or higher, enable mod_rewrite, if you want to use MySQL we prefer 5.5.27 version.

Also prefer to use xampp


Set up a new project

You must open ./application/config/config.php and update : 

<?php 
/** 
 * Project 
 * 
 * @var string $_config['project']['url'] Project full URL 
 */ 
$_config['project']['url']     = 'http://localhost/cliprz'; 
?>
to your project full URL.


Browse project

To browse project go to http://localhost/projectname/home ,You will get a Cliprz framework main home page, to edit this page go to ./application/project/home/home.view.php file, you can edit controller on ./application/project/home/home.controller.php. there is a static files in ./application/project/static/ as header.view.php and footer.view.php.
